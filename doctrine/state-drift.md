# State Drift

State drift occurs when a system gradually moves away from reality while continuing to operate correctly within its own internal logic.

A system can:

- produce consistent outputs  
- pass validation checks  
- remain internally coherent  

…and still become increasingly misaligned with reality.

This happens because:

👉 the system reinforces its own state over time

Feedback does not always correct the system.

👉 it can stabilize and amplify incorrect state

---

## Key Insight

Systems do not fail when they break.

👉 They fail when they stabilize around the wrong state.
